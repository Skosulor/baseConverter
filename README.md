# Number Base Converter

A simple web-based tool to convert numbers between different bases and representations.

## Features

- **Convert between**: Decimal, Binary, Hexadecimal, and Base64
- **Two's Complement**: View 8-bit, 16-bit, 32-bit, and 64-bit representations
- **Signed interpretation**: Automatically shows signed decimal value when entering binary/hex
- **ASCII**: Displays ASCII character for values 32-126
- **Real-time conversion**: Type in any field and see live updates

## Usage
```bash
cd frontend
npm install
npm run dev
```
